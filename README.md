# crac-amateur-radio-exam-questions-2025-csv
将于2025年10月1日生效的[《新版业余无线电台操作技术能力验证题库（2025年版）》](https://mp.weixin.qq.com/s/abWyWXJgkknclMqQG87MrA "微信公众号文章")，转换为了CSV格式，以方便制作模拟考试工具。

## 文件描述
| 文件名 | 描述 |
|------|------|
|class_a.csv|A类题库|
|class_b.csv|B类题库|
|class_c.csv|C类题库|
|full.csv|总题库|
|images.csv|题目附图与题号对照表|
|images/*.jpg|题目附图|

## 特别说明
个别题目有2个题号，使用半角逗号分割。
